Code to reset configurations on Pods in the LAN Lab

How to use (Windows):
# Setup

## Create Virtual Environment

```powershell
python -m venv .venv
```

## Activate Virtual Environment

```powershell
.\.venv\Scripts\Activate.ps1
```

## Install Dependencies

```powershell
pip install -r requirements.txt
```

## Run Script

```powershell
python reset_pod.py
```
